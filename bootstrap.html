Bootstrap Framework

Bootstrap is the world's most popular open-source CSS framework. It provides a responsive 12-column grid system, pre-styled UI components, and utility classes that enable developers to build professional, mobile-first layouts rapidly.

The 12-Column Grid

Bootstrap's grid system divides the page into 12 columns. Using classes like col-md-4 (4 columns on medium screens) or col-lg-6, you can define exactly how many columns each element occupies at each breakpoint — automatically collapsing to full width on mobile.

Utility Classes

Bootstrap's utility system (mt-3, d-flex, text-center, bg-primary) lets you apply common styles directly in HTML without writing any custom CSS, drastically speeding up development.

Pre-built Components

Bootstrap ships with dozens of ready-made components — navbars, cards, modals, accordions, carousels, alerts, badges, and more — all styled consistently and accessible out of the box.
