What this page demonstrates

This page is built with pure HTML5 only — no CSS, no JavaScript. It demonstrates the following HTML concepts:

  • Semantic landmark elements: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • Accessible form with a <fieldset> and <legend>
  • Correct <label for> / <input id> pairing
  • HTML5 input types: email, password, checkbox
  • Native validation: required, minlength, type="email"
  • Accessibility attributes: aria-describedby, aria-required, aria-live
  • Metadata: autocomplete, inputmode
  • Semantic text: <strong>, <em>, <time>, <abbr>, <code>

Key HTML Elements Used

Semantic HTML5 Elements Demonstrated on This Page
Element Purpose
<header> Introductory content for the page or section
<nav> Navigation links — labelled with aria-label
<main> The primary unique content of the page (used once)
<section> Thematic grouping of content, each labelled by heading
<article> Self-contained content (the form widget)
<fieldset> / <legend> Groups related form controls with an accessible label
<label for="…"> Associates a text label with an input for accessibility
<footer> Footer information for the page
