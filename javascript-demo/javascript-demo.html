JavaScript Calculator

Demonstrates DOM manipulation, event handling, and state management — vanilla JS only.

JavaScript Concepts Demonstrated

1. VARIABLES & STATE

Three let variables hold the calculator's state: currentInput (what you're typing), previousValue (the stored number), and operator (the pending operation). State persists across function calls.

2. DOM SELECTION

document.getElementById('display') and querySelector() grab references to HTML elements. The script reads and writes their textContent to update what the user sees in real time.

3. EVENT HANDLING

Each button uses onclick attributes to call JavaScript functions. The keyboard listener uses document.addEventListener('keydown', handler) to map physical keys to calculator actions.

4. FUNCTIONS & CONTROL FLOW

Each action (appendDigit(), setOperator(), calculateResult()) is an isolated function. A switch statement inside calculateResult() selects the correct arithmetic operation.

5. ERROR HANDLING

Division by zero is caught explicitly: if (b === 0) return 'Error'. The isNaN() guard prevents invalid parse results from propagating through the display.

6. CHAINED OPERATIONS

If you press an operator when one is already pending, calculateResult() is called first to compute the intermediate result — enabling chains like 5 + 3 × 2 without losing the previous value.

⌨️ Keyboard Support

0–9 digits · + - * / operators · Enter or = to calculate · Backspace to delete · Escape to clear
