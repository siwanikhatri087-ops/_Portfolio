All styling achieved with pure CSS only — no JavaScript. Hover over the cards to see transitions.

🎨

CSS Custom Properties

All colours and values are defined as --variables in :root, enabling a consistent, maintainable theme across the entire stylesheet.

📐

CSS Grid Layout

This grid uses auto-fit + minmax(260px, 1fr) — cards automatically reflow from 3 columns on desktop down to 1 on mobile, with no media queries.

🪟

Glassmorphism

The frosted-glass effect uses backdrop-filter: blur() on a semi-transparent rgba() background — a modern CSS UI pattern.

✨

CSS Animations

Cards fade in with staggered animation-delay on each :nth-child. The background gently shifts hue using an infinite @keyframes loop.

🖱️

Hover Transitions

Cards smoothly lift with transform: translateY(-10px) on :hover, using transition for a polished, eased movement — no JavaScript required.

📱

Responsive Design

The layout adapts to any viewport using CSS Grid's auto-fit, clamp() for fluid font sizes, and min() for a responsive container width.
