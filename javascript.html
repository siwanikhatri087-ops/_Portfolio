Section 5

JavaScript

JavaScript is the programming language of the web. It runs directly in the browser and enables dynamic, interactive experiences — responding to user actions, manipulating the DOM, fetching data asynchronously, and driving complex application logic.

DOM Manipulation

The Document Object Model (DOM) is JavaScript's interface to the HTML document. Using methods like document.getElementById(), querySelector(), and properties like textContent / innerHTML, JavaScript can read and modify any element on the page in real time.

Event Handling

JavaScript listens for user events — clicks, keyboard presses, form submissions, mouse movements — using addEventListener(). When an event fires, the associated callback function runs, making the page respond dynamically to user interaction.

State & Application Logic

Variables hold application state — the current values a program is tracking (e.g., the running total in a calculator). Functions encapsulate logic and can be called repeatedly. Together they form the foundation of interactive apps without needing a framework.

JavaScript ES6 
let count = 0;
const display = document
  .querySelector('#counter');

function increment() {
  count++;
  display.textContent = count;
}

document
  .querySelector('#btn')
  .addEventListener('click', increment);
